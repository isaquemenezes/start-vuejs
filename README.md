```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

### Router
1. cli
    `vue add router`
2. npm 
    `npm install vue-router@4`

### VUEX -state
1. cli 
    `vue add vuex`

2. npm 
    `npm install vuex@next --save`
